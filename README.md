# x_team_procgen_pw

X-Team Procedural Generation example.

# Installation and Set Up

Setup instructions:
```bash
    $ npm install
    $ npm run start
```

Views:
```
    http://localhost:7777/potion
    http://localhost:7777/tiles
    http://localhost:7777/automata
```
